## 校准与覆盖｜2027年上半年（Q1–Q2）

- [ ] **持续校准闭环**
  - [ ] 按分组与时间尺度，将 QQ 评分与实际表现进行 <span className="pink-glow-text">对照</span>  
  - [ ] 在严格版本管理与可回滚机制下，调优权重、阈值与方法论  
  - [ ] 跟踪不同市场状态下的稳定性，包括牛市/熊市，以及高/低波动环境

- [ ] **数据可靠性与覆盖范围**
  - [ ] 扩展覆盖范围必须通过 <span className="pink-glow-text">质量闸门</span>：新鲜度、置信度与异常过滤  
  - [ ] 展示指标置信度，使策略可要求达到最低质量门槛


## 个人 QQ OS｜2027年下半年（Q3–Q4）

- [ ] **投资组合与关注列表叠加层**
  - [ ] 用户输入持仓与关注列表  
  - [ ] QQ 输出聚合后的 <span className="pink-glow-text">风险画像</span> 与关键红旗信号  
  - [ ] 将用户目标转化为组合规则，并生成 <span className="pink-glow-text">配置与再平衡方案</span>  
  - [ ] 强制执行护栏，包括风险预算、敞口上限、以及流动性与滑点约束

- [ ] **可解释性**
  - [ ] 自动生成 <span className="pink-glow-text">“为什么会这样”</span> 分析：包括 PnL 归因与事件轨迹  
  - [ ] 决策追踪：展示哪些 QQ 信号触发了哪些动作
