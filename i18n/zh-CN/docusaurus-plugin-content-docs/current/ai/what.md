## 什么是智能体？

在 QQ Omega 中，**智能体** 是一个“小型专家”。  
你可以把它想象成 **只做一件事的专科医生**：有人只看心脏，有人只看肺部，有人只负责读化验单。

每个智能体都会：
- 只关注 <span className="pink-glow-text">**一个极其具体的领域**</span>（例如团队可信度、代币解锁、巨鲸资金流、央行利率路径）；  
- 读取 **结构化数据与量化指标**；  
- 输出可被整合的 **分数、信号或注释**。

单个智能体并不试图理解一切。  
它的目标是：在 **一个点上做到极强**。

## 什么是集群？

**智能体集群** 是一支 <span className="pink-glow-text">**围绕同一主题协作的智能体团队**</span>：它们研究同一类问题，但从不同角度切入。

你可以把它想象成：
- 医院里的 **一个科室**：很多医生，一个领域；  
- 或同一部门的 **分析师团队**：每个人有自己的模型，但研究的是同一类资产与同一类问题。

它们使命一致，但每个智能体都有自己的数据集、规则体系与评分方法。

## 元素 = 智能体集群

在 QQ Omega 里，每一个维度（基本面、代币经济等）都不是“单一模块”，而是一个完整的 **智能体集群**：由大量窄领域专家智能体组成，协同产出可比较的结论。

每个维度对应一个 <span className="pink-glow-text">元素</span>：


| 集群 | 领域 |
|:------:|-------------|
| <span className="pink-badge">[**碳集群**](carbon.md)</span> | **基本面** – 团队、产品、商业模式、治理、安全。 |
| <span className="pink-badge">[**金集群**](gold.md)</span> | **代币经济** – 供给、通胀、销毁、解锁、回购、收益。 |
| <span className="pink-badge">[**氢集群**](hydrogen.md)</span> | **链上资金流** – TVL、成交量、持仓结构、吸筹 vs 派发。 |
| <span className="pink-badge">[**硅集群**](silicon.md)</span> | **技术与量化** – 价格结构、指标、清算、状态识别。 |
| <span className="pink-badge">[**氧集群**](oxygen.md)</span> | **宏观与市场状态** – 利率、流动性、主导率、市值、情绪。 |
| <span className="pink-badge">[**磷集群**](phosphorus.md)</span> | **统筹与聚合** – 分项分数、QQ 总评分、最终洞察。 |



因此：
- **智能体** = 一个只做一件事的专业小专家。  
- **智能体集群** = 一组围绕同一主题协作的智能体。  
- **元素（碳、金、氢、硅、氧、磷）** = **一个具备明确任务边界的主题集群**。

QQ Omega 依赖 **多个专业集群** 同时工作：每个集群内部有多个智能体，最后把它们的观点整合为一幅单一且 <span className="pink-glow-text">**自洽**</span> 的全景图。
