## 什么是 QQ Omega？

QQ Omega 是一套高精度评分引擎：把 <span className="pink-glow-text">量化与定性情报</span> 融合为一个统一框架，用于对加密资产进行一致、可比较的评估。

你可以把 QQ Omega 理解为你在加密资产领域的 <span className="pink-glow-text">智能“右手”</span>：它是一层持续更新的分析体系，目标是在全球最剧烈波动的市场中，为你提供更清晰、更结构化的判断基础。

QQ Omega 会在 <span className="pink-glow-text">五个类别</span> 上评估每一种加密资产：**基本面、代币经济、链上行为、技术结构、以及宏观与市场状态**，并在多个时间尺度下给出结果。每一个类别背后都是一个由多名“窄领域专家”组成的 **智能体集群**：每个智能体只负责一小块分析。若想深入了解每个类别背后的结构，请参阅 <span className="pink-badge">[智能体集群](/docs/category/ai-agent-swarms)</span>。

这些分析会被聚合为一个全局 <span className="pink-glow-text">QQ 评分</span>：为每个加密资产生成 0 到 100 的分数，并在新数据与新事件出现时 <span className="pink-glow-text">自动更新</span>。

系统还会通过对不同智能体赋予不同权重，生成面向不同时间尺度的多套 QQ 评分：短线评估会更强调链上数据与技术结构，而长线评估会更重视基本面与代币经济。

最终，你会得到按时间尺度划分的独立排名，从而判断某个资产在你的投资周期内，看起来是更 <span className="pink-glow-text">低估</span> 还是更 <span className="pink-glow-text">高估</span>。

QQ Omega 提供并列对照的视图：结构性强弱、市场过热程度、以及由叙事与热度驱动的行为差异——把复杂性压缩成可解释的评分与排名，支持更清醒、更高确信度的投资决策。

:::note
把 QQ Omega 当作一个 <span className="pink-glow-text">**量化 + 定性的一体化加密分析师**</span>：它吸收数据、建立结构，并将复杂性压缩为 **清晰的评分、排名与可解释结论**，作为你做出 **有意识投资决策** 的底层依据。
:::
