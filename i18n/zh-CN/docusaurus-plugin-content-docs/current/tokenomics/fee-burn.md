### 运作方式

协议金库收到的 LP 手续费，会按照下述分配规则处理。

### 销毁分配
<span className="pink-glow-text">**LP 手续费的 12%**</span> 会被分配到 $QQ 的 <span className="pink-glow-text">**销毁**</span> 机制中。这部分资金用于永久销毁 $QQ：将代币发送至“黑洞地址”（不可用地址），从而随时间推移降低流通供给。

### POL 分配（协议自有流动性）
<span className="pink-glow-text">**LP 手续费的 17%**</span> 会被分配至 <span className="pink-glow-text">**POL（协议自有流动性）**</span>。POL 指协议使用手续费，将 $QQ 与配对资产一起补充为流动性，从而长期支撑 **Solana** 与 **Base** 上的 $QQ 流动性池。

### 销毁规则
销毁流程会持续进行，直到总供应量达到 <span className="pink-glow-text">**???.???.???**</span> $QQ。最终供给并非预先固定，而取决于协议随时间产生的真实活动；但该机制被设计为至少移除 <span className="pink-glow-text">**初始总供应量的 50%**</span>。

### 每月执行与透明披露
协议分配会以 <span className="pink-glow-text">按月节奏</span> 通过链上交易透明执行，并汇总上一个周期的全部活动。每个月，QQ Omega 会发布公开回顾，提供每一笔协议级资金动作的 **链上凭证**，使任何人都能独立验证资金如何被使用、转移，或从流通中移除。
