### 关于 $QQ 空投计划

$QQ 空投是一个持续进行、跨多个赛季（Season）的长期计划，用于在时间维度上奖励生态早期参与者。

:::note
- 不要期待会发布完整的“如何获得资格”指南（可能会出现少量提示）。  
- **不保证**任何具体行动、行为或贡献一定会获得奖励。  
:::

### 分配比例

- <span className="pink-glow-text">总供应量的 6%</span> 预留给 <span className="pink-glow-text">Virtuals 生态</span>，将在 <span className="pink-glow-text">多个</span> 赛季与多条奖励路径中分发。

- <span className="pink-glow-text">总供应量的 15%</span> 预留给 <span className="pink-glow-text">QQ 社区</span>，将在 <span className="pink-glow-text">多个</span> 赛季与多条奖励路径中分发。
