## How QQ Scores & AI Agents Work?

:::note
TLDR: **Data → AI agents → Scores → Clear overview for your decisions.**
:::

1. **Data collection**  
   QQ Omega collects and organizes on-chain data such as holders, flows, liquidity, and token activity, as well as off-chain data such as products, documentation, news, and macroeconomic context.

2. **AI agents per dimension**  
   Data is grouped by dimensions and fed into **multiple, structured <span className="pink-glow-text">swarms of AI agents</span>**, each dedicated to a specific analytical area.

   Here are some examples:
   - **Fundamentals swarm**: one agent for team, one for product, one for security, and more.
   - **Tokenomics swarm**: one agent for buybacks, one for emissions, one for unlocks, one for utility, and more.

3. **From swarms to scores**
- Each swarm produces a <span className="pink-glow-text">**dimension score**</span> based on its own metrics and evaluation rules (for example, a Fundamentals Score).
- All dimensions scores are combined using **different weights and normalization methods** into a <span className="pink-glow-text">**global QQ score**</span> for each crypto-asset.
- Each crypto-asset has <span className="pink-glow-text">**four global QQ scores**</span>: <span className="pink-glow-text">Daily</span>, <span className="pink-glow-text">Weekly</span>, <span className="pink-glow-text">Monthly</span>, and <span className="pink-glow-text">Yearly</span>, so the score matches the **time horizon** of the data and the **weights** used across dimensions.

4. **Continuous updates**  
   As new data becomes available, such as on-chain movements, token unlocks, new partnerships, or shifts in market sentiment, agents update their assessments and the scores <span className="pink-glow-text">**evolve over time**</span>.
